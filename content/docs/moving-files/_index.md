title: Moving Files
---

Use **uploads** for quick or governed transfers; use **streaming** for
large files or unreliable networks (with resume support). Both are
policy-governed and auditable.

- [File uploads](/docs/moving-files/uploading-files/) — What uploads
  are, when to use them, and how they work (user and admin)
- [File streaming](/docs/moving-files/streaming-files/) — What
  streaming is, when to use it, and how it works (user and admin)

For step-by-step send/receive instructions, see [Transfer a
file](/docs/guides/transfer/).
